# Mindfulness App Product Analytics

## Overview
This project analyzes user behavior, engagement, and retention in a mindfulness mobile application. Using SQL, Python, and Power BI, the work diagnoses funnel drop-offs and retention patterns and proposes a data-backed product change to improve early retention and premium conversion.

## Tools
- SQL (funnel and cohort analysis)
- Python (data cleaning, validation, visualization)
- Power BI (stakeholder dashboard)
- Markdown (product recommendations and PRD-lite)

## Dataset
The dataset is synthetic and simulates users, sessions, events, and content interactions inside a mindfulness mobile application.

## Project Structure
- `data_raw/`: original datasets (unchanged)
- `data_clean/`: cleaned and derived datasets
- `sql/`: analysis queries
- `notebooks/`: Python notebooks
- `powerbi/`: dashboard files
- `writeup/`: product framing and recommendations
- `figures/`: saved visuals for reports and presentations
